# 숏폼 크리에이터 선정 기획 스킬

숏폼 캠페인에 적합한 크리에이터를 체계적으로 선정하는 기준과 전략을 수립하는 스킬입니다.

## 의존 도구 및 스킬

**필수 확장:**
- `ask`: 사용자에게 객관식 질문 (단일)
- `ask_multi`: 사용자에게 여러 객관식 질문 (순차적)

**연계 스킬:**
- `content-planner`: 컨텐츠 기획 완료 후 사용 권장
- `hook-planner`: 후킹 기획 참고

---

## 프로세스

### 1단계: 캠페인 정보 확인

> 💡 `content-planner` 스킬을 먼저 실행했다면 해당 정보를 재활용합니다.

**필요한 정보:**
- 대상 국가
- 주요 플랫폼 (TikTok / Instagram Reels / YouTube Shorts)
- 타겟 오디언스 (연령대, 성별)
- 리워드 수준
- 캠페인 목표 (인지도 / 전환 / 브랜딩)
- 컨텐츠 방향 (기획된 컨텐츠 유형들)

정보가 없으면 `ask_multi`로 수집:

```
questions: [
  { key: "country", question: "대상 국가는?", options: ["한국", "미국", "일본", "동남아", "유럽"] },
  { key: "platform", question: "주요 플랫폼은?", options: ["TikTok", "Instagram Reels", "YouTube Shorts"] },
  { key: "target_age", question: "타겟 연령대는?", options: ["10-20대", "20-30대", "30-40대", "40-50대", "전 연령"] },
  { key: "target_gender", question: "타겟 성별은?", options: ["여성", "남성", "전체"] },
  { key: "reward", question: "리워드 수준은?", options: ["제품만 제공", "제품 + 소액 (10만원 미만)", "제품 + 중액 (10-30만원)", "제품 + 고액 (30만원+)"] },
  { key: "goal", question: "캠페인 목표는?", options: ["인지도 향상", "전환 유도", "브랜드 이미지 구축"] }
]
```

### 2단계: 제품/브랜드 특성 확인

> 💡 **ask 확장 사용**

**2-1. 제품 카테고리**
```
question: "제품 카테고리는?"
options: ["뷰티/화장품", "패션/액세서리", "F&B/식품", "테크/가전", "웰니스/건강", "생활용품", "기타"]
```

**2-2. 브랜드 톤앤매너**
```
question: "브랜드 톤앤매너는?"
options: ["프리미엄/럭셔리", "트렌디/영", "자연/클린", "전문가/신뢰", "펀/유머러스", "미니멀/모던"]
```

**2-3. 컨텐츠 제한사항** (직접 입력)
```
"크리에이터 선정 시 피해야 할 요소가 있나요?"
예시: "의료 효과 주장하는 크리에이터", "경쟁사 협찬 이력", "특정 정치적 발언"
```

### 3단계: 컨텐츠별 크리에이터 유형 매칭

기획된 컨텐츠 유형에 따라 적합한 크리에이터 유형을 매칭합니다.

#### 컨텐츠 유형 → 크리에이터 유형 매핑

| 컨텐츠 유형 | 추천 크리에이터 유형 | 핵심 역량 |
|------------|---------------------|----------|
| **리뷰/설명형** | 뷰티 리뷰어, 테크 리뷰어 | 설득력 있는 설명, 신뢰감 |
| **언박싱/ASMR** | 언박싱 전문, ASMR 크리에이터 | 시각적 연출, 사운드 |
| **튜토리얼** | How-to 크리에이터, 전문가 | 교육적 컨텐츠 제작 |
| **라이프스타일/루틴** | 일상 브이로거, 웰니스 크리에이터 | 자연스러운 일상 연출 |
| **비포애프터** | 변화 컨텐츠 전문 | 극적인 비교 연출 |
| **챌린지/트렌드** | 댄서, 트렌드세터 | 트렌드 민감도, 바이럴 |
| **스타일링/OOTD** | 패션 인플루언서 | 스타일링 센스 |
| **먹방/푸드** | 푸드 크리에이터 | 맛 표현, 먹방 |

---

### 4단계: 국가별 크리에이터 특성 반영

#### 국가별 크리에이터 선정 기준

| 국가 | 선호 크리에이터 스타일 | 주의사항 |
|------|----------------------|----------|
| **한국** | 전문성 + 친근함, 디테일한 리뷰 | 광고 표시 필수, 뒷광고 민감 |
| **미국** | 다양성, 진정성, 개성 강한 | 다양한 인종/체형 포함 권장 |
| **일본** | 정제된 컨텐츠, 상세 설명 | 과도한 과장 지양 |
| **동남아** | 높은 에너지, 가성비 강조 | 로컬 언어 사용 크리에이터 |
| **유럽** | 미니멀, 에스테틱, 지속가능성 | 환경/윤리 의식 높음 |

---

### 5단계: 리워드별 크리에이터 티어 결정

#### 리워드 수준 → 크리에이터 티어

| 리워드 수준 | 추천 팔로워 규모 | 배분 전략 |
|------------|-----------------|----------|
| **제품만 제공** | Nano (1K-10K) 70%, Micro (10K-50K) 30% | 물량 중심, 진성 리뷰 |
| **제품 + 소액** | Micro (10K-50K) 60%, Nano 40% | 참여율 중심 |
| **제품 + 중액** | Micro 50%, Mid-tier (50K-200K) 50% | 도달+참여 균형 |
| **제품 + 고액** | Mid-tier 60%, Macro (200K+) 40% | 도달 중심, 브랜딩 |

#### 팔로워 규모별 특성

| 티어 | 팔로워 수 | 평균 참여율 | 특징 | 비용 효율 |
|------|----------|------------|------|----------|
| **Nano** | 1K-10K | 8-15% | 높은 진정성, 니치 타겟 | ⭐⭐⭐⭐⭐ |
| **Micro** | 10K-50K | 4-8% | 참여+도달 균형, 전문성 | ⭐⭐⭐⭐ |
| **Mid-tier** | 50K-200K | 2-4% | 넓은 도달, 브랜드 인지도 | ⭐⭐⭐ |
| **Macro** | 200K-1M | 1-2% | 대규모 도달, 브랜딩 | ⭐⭐ |
| **Mega** | 1M+ | 0.5-1% | 최대 노출, 화제성 | ⭐ |

### 6단계: 크리에이터 선정 기준 수립

#### 필수 조건 체크리스트

| # | 기준 | 확인 방법 |
|---|------|----------|
| 1 | **국가/언어 일치** | 프로필, 컨텐츠 언어 확인 |
| 2 | **타겟 연령대 도달** | 오디언스 분석 (Creator Marketplace) |
| 3 | **참여율 기준 충족** | 최근 30개 게시물 평균 계산 |
| 4 | **브랜드 세이프티** | 과거 컨텐츠 스캔, 논란 이력 확인 |
| 5 | **경쟁사 협찬 이력** | 최근 6개월 협찬 게시물 확인 |
| 6 | **컨텐츠 퀄리티** | 영상 편집, 조명, 사운드 품질 |

#### 참여율 기준 (플랫폼별)

| 플랫폼 | 최소 참여율 | 우수 참여율 |
|--------|-----------|-----------|
| **TikTok** | 4% | 8%+ |
| **Instagram Reels** | 3% | 6%+ |
| **YouTube Shorts** | 2% | 5%+ |

> 참여율 = (좋아요 + 댓글 + 공유) / 조회수 × 100

---

### 7단계: 크리에이터 페르소나 정의

각 컨텐츠 유형에 맞는 이상적인 크리에이터 페르소나를 정의합니다.

```
### 크리에이터 페르소나 A: [컨텐츠 유형]

**프로필:**
- 팔로워: {규모}
- 연령대: {크리에이터 본인 연령}
- 성별: {성별}
- 주요 컨텐츠: {컨텐츠 유형}
- 톤앤매너: {스타일}

**필수 조건:**
- [ ] {조건 1}
- [ ] {조건 2}
- [ ] {조건 3}

**우대 조건:**
- {우대 1}
- {우대 2}

**검색 키워드:**
- #{해시태그1} #{해시태그2} #{해시태그3}

**레퍼런스 크리에이터:** (유사한 스타일 예시)
- @{계정명}: {이유}
```

### 8단계: 피해야 할 크리에이터 정의

#### 공통 제외 기준

| 유형 | 제외 이유 |
|------|----------|
| **가짜 팔로워 의심** | 팔로워 대비 참여율 극히 낮음 (< 0.5%) |
| **논란/스캔들 이력** | 브랜드 리스크 |
| **과도한 협찬 비율** | 광고 피로도, 신뢰도 저하 |
| **경쟁사 최근 협찬** | 메시지 혼란, 브랜드 충돌 |
| **컨텐츠 품질 저하** | 저화질, 편집 부족 |
| **응답률 낮음** | 협업 진행 어려움 |

#### 제품/브랜드별 추가 제외 기준

```
예시 (웰니스 제품):
- ❌ 의료 효과를 과장하는 크리에이터
- ❌ 헬스케어/의료 전문 크리에이터 (포지셔닝 충돌)
- ❌ 다이어트/체중 감량 중심 크리에이터

예시 (럭셔리 브랜드):
- ❌ 저가/가성비 중심 크리에이터
- ❌ 과도한 유머/밈 스타일
```

### 9단계: 아웃리치 전략 수립

#### 리워드별 아웃리치 메시지 톤

| 리워드 | 메시지 톤 | 강조 포인트 |
|--------|----------|------------|
| **제품만** | 겸손, 협력 제안 | 제품 가치, 크리에이터 스타일 존중 |
| **제품 + 소액** | 파트너십 제안 | 상호 이익, 장기 협력 가능성 |
| **제품 + 중/고액** | 프로페셔널 | 명확한 조건, 기대 성과 |

#### 아웃리치 체크리스트

- [ ] 크리에이터 최근 컨텐츠 언급 (개인화)
- [ ] 브랜드/제품 소개 간결하게
- [ ] 리워드 조건 명확히
- [ ] 컨텐츠 자유도 명시 (가이드라인 수준)
- [ ] 응답 기한 설정
- [ ] 연락처/담당자 정보

## 파일 저장

- 기본 경로: `./[브랜드명]_크리에이터_선정_가이드.md`

## 연계 스킬 사용 흐름

```
1. content-planner → 컨텐츠 기획
2. hook-planner → 후킹 기획
3. creator-planner → 크리에이터 선정 (현재 스킬)
```

> 💡 `content-planner`에서 기획된 컨텐츠 정보를 기반으로 크리에이터를 선정하면 더 정확한 매칭이 가능합니다.
