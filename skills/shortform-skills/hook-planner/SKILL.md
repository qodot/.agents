# 숏폼 첫 3초 후킹 기획 스킬

숏폼 컨텐츠의 성패를 결정하는 **첫 3초 후킹**을 체계적으로 기획하는 스킬입니다.

## 의존 도구 및 스킬

**필수 확장:**
- `ask`: 사용자에게 객관식 질문 (단일)
- `ask_multi`: 사용자에게 여러 객관식 질문 (순차적)

**연계 스킬 (shortform-skills 내):**
- `content-planner`: 컨텐츠 기획 (이 스킬에서 hook-planner 호출)
- `creator-planner`: 크리에이터 선정 기획

---

## 프로세스

### 1단계: 제품 정보 수집

> 💡 **ask 확장 사용**

**1-1. 제품 카테고리**
```
question: "제품 카테고리는?"
options: ["스킨케어", "색조 화장품", "헤어케어", "식품/음료", "패션/액세서리", "테크/가전", "웰니스/건강", "생활용품"]
```

**1-2. 가격대**
```
question: "제품 가격대는?"
options: ["저가 (2만원 미만)", "중저가 (2-5만원)", "중가 (5-15만원)", "고가 (15만원 이상)"]
```

**1-3. 제품 USP** (직접 입력)
```
"제품의 핵심 USP(차별점)를 알려주세요"
예시: "3600 Gauss 네오디뮴 자석", "비건 인증", "24시간 지속력"
```

**1-4. 제한사항** (직접 입력)
```
"마케팅 시 금지되는 표현이나 제한사항이 있나요?"
예시: "의료 효과 언급 금지", "경쟁사 비교 금지", "없음"
```

### 2단계: 타겟 정보 수집

> 💡 **ask_multi 확장 사용**

```
questions: [
  { 
    key: "country", 
    question: "대상 국가/지역은?", 
    options: ["한국", "미국", "일본", "동남아", "유럽", "글로벌"] 
  },
  { 
    key: "age", 
    question: "주요 타겟 연령대는?", 
    options: ["Z세대 (10-20대)", "밀레니얼 (30대)", "X세대 (40대)", "전 연령대"] 
  },
  { 
    key: "gender", 
    question: "주요 타겟 성별은?", 
    options: ["여성", "남성", "전체 (젠더리스)"] 
  }
]
```

**2-4. 타겟 페인포인트** (직접 입력)
```
"타겟이 겪는 문제/고민은?"
예시: "선크림 백탁", "액세서리가 건강용품처럼 생겼음", "가격이 비쌈"
```

### 5단계: 후킹 유형 결정

수집된 정보를 바탕으로 적합한 후킹 유형을 결정합니다.

#### 5-1. 국가별 후킹 스타일 매칭

| 국가 | 기본 스타일 | 톤앤매너 |
|------|------------|----------|
| **한국** | 공감형, 트렌드, 질문형 | 친근, 밈 활용, "~하는 거 나만 몰랐어?" |
| **미국** | 직접적, 대담, 명령형 | 자신감, "Stop scrolling", "This is THE..." |
| **일본** | 정중, 결과 예고형 | 상세, "〜してみた結果...", "正直レビュー" |
| **동남아** | 리액션, 가성비 | 과장, "OMG!", "Only $XX!" |
| **유럽** | 미니멀, 에스테틱 | 세련, 비주얼 중심 |

#### 5-2. 연령대별 후킹 스타일 매칭

| 연령대 | 선호 스타일 | 키워드 |
|--------|------------|--------|
| **Z세대** | 밈, 자기비하, 빠른 템포 | POV, 챌린지, 트렌드 음악 |
| **밀레니얼** | 공감, 실용, 라이프스타일 | 루틴, 꿀팁, 가성비 |
| **X세대** | 신뢰, 전문가, 상세 | 검증, 추천, 효과 |
| **전 연령** | 보편적 공감 + 시각적 | 디자인, 일상, 루틴 |

#### 5-3. 카테고리별 효과적인 후킹 패턴

| 카테고리 | 추천 후킹 유형 | 예시 |
|----------|---------------|------|
| **스킨케어** | B/A, 기간, 성분 | "7일 후 피부 변화" |
| **색조** | 커버력, 발색, 트랜지션 | "한 번에 커버되는 거 실화?" |
| **식품** | 맛, 리액션, 먹방 | "이 맛 뭐야..." |
| **패션/액세서리** | 스타일링, OOTD | "이 조합 찾았다" |
| **웰니스** | 루틴, 라이프스타일 | "매일 하는 습관" |
| **테크** | 기능, 놀라움 | "이게 된다고?" |

#### 5-4. 캠페인 목적별 후킹 방향

| 목적 | 후킹 방향 | 특징 |
|------|----------|------|
| **바이럴** | 공유 유도, 밈, 호기심 | 결말 안 보여줌 |
| **전환** | 혜택, 긴급성, CTA | "오늘까지", "링크" |
| **브랜딩** | 스토리텔링, 감성 | 브랜드 가치 전달 |

### 6단계: 후킹 초안 생성

위 분석을 종합하여 **5가지 후킹 초안**을 생성합니다.

각 후킹은 다음 형식으로 제시:

```
### 후킹 A: [유형명]

**텍스트/멘트:**
> "{후킹 문구}"

**썸네일/첫 장면:**
{시각적 연출 설명}

**왜 이 후킹인가:**
- {국가} 타겟에게 {이유}
- {연령대}가 선호하는 {스타일}
- {카테고리}에서 효과적인 {패턴}

**강도 예상:** ⭐⭐⭐⭐ (4/5)
```

---

### 7단계: 후킹 강도 검증

각 후킹에 대해 체크리스트로 강도를 검증합니다.

| # | 체크 항목 | 설명 |
|---|----------|------|
| 1 | **3초 규칙** | 3초 안에 핵심 전달? |
| 2 | **스크롤 스톱** | 시각적으로 멈출 요소? |
| 3 | **궁금증** | "다음이 뭐지?" 유발? |
| 4 | **공감** | "나도 그래!" 포인트? |
| 5 | **제한사항** | 금지 표현 위반 없음? |
| 6 | **타겟 적합** | 해당 국가/연령에 맞음? |
| 7 | **USP 연결** | 제품 차별점과 연결? |

**점수:** 7개 중 몇 개 통과?
- 7개: ⭐⭐⭐⭐⭐ (완벽)
- 5-6개: ⭐⭐⭐⭐ (좋음)
- 3-4개: ⭐⭐⭐ (보통)
- 2개 이하: 재검토 필요

### 8단계: 최종 선택 & 개선

> 💡 **ask 확장 사용**

```
question: "어떤 후킹이 가장 마음에 드시나요?"
options: ["후킹 A", "후킹 B", "후킹 C", "후킹 D", "후킹 E", "조합/변형 원함"]
```

선택된 후킹에 대해:
1. **변형 버전** 2-3개 추가 제안
2. **썸네일 상세 연출** 구체화
3. **예시 멘트** 다듬기

## 후킹 유형 레퍼런스

### 유형 1: 질문형
```
"아직도 ___로 고민해?"
"이거 나만 몰랐어?"
"왜 아무도 안 알려줬지?"
```

### 유형 2: 명령형
```
"Stop scrolling"
"Put your phone down and watch this"
"이거 끝까지 봐"
```

### 유형 3: 충격형
```
"___명이 이미 바꿨다"
"이거 실화야?"
"믿기 힘들겠지만..."
```

### 유형 4: 비교형
```
"Before vs After"
"이거 vs 저거"
"1일차 vs 7일차"
```

### 유형 5: 스토리형
```
"처음엔 나도 안 믿었어"
"솔직히 말할게..."
"드디어 찾았다"
```

### 유형 6: POV/트렌드형
```
"POV: you just discovered..."
"POV: 늦잠 잔 아침"
"This is your sign to..."
```

### 유형 7: 시각형 (텍스트 최소)
```
- 제품 쏟아붓기
- 극적인 클로즈업
- ASMR 사운드
- 화면 전환
```

## 기존 기획서 참고

후킹 기획 전, **유사한 카테고리/국가의 기존 기획서**를 참고하여 후킹 스타일과 톤앤매너를 맞춥니다.

### 참고 기획서 목록

기획서는 `../examples/` 폴더에 위치합니다.

| 파일명 | 카테고리 | 참고 포인트 |
|--------|----------|-------------|
| `[메디필_미국] 바이럴 영상 기획안.pdf` | 화장품 (스킨케어) | 미국 시장 후킹 스타일 |
| `[웰라쥬] 바이럴 영상 기획안_4차.pdf` | 화장품 (스킨케어) | 정제된 후킹 문구 |
| `[이니스프리] 바이럴 영상 기획안.pdf` | 화장품 (스킨케어) | 대형 브랜드 후킹 |
| `[비비드로우] 바이럴 기획안.pdf` | 화장품 (색조) | 색조 제품 후킹 |
| `[더후] 선라인 기획안.pdf` | 화장품 (럭셔리) | 럭셔리 톤앤매너 |
| `[아비브] 바이럴 기획안.pdf` | 화장품 (스킨케어) | 스킨케어 후킹 |
| `[블랑두부] 콘텐츠 기획안.pdf` | F&B (식품) | 식품 카테고리 후킹 |
| `[에스쁘아] 1차 - Be velvet Cover Cushion.pdf` | 화장품 (색조) | 쿠션 제품 후킹 |
| `[불스원] 2차 기획안.pdf` | 자동차용품 | 비화장품 카테고리 참고 |

### 참고 방법

1. **카테고리 매칭**: 현재 작업 중인 브랜드와 유사한 카테고리의 기획서 선택
2. **PDF 읽기**: `read` 도구로 해당 PDF 파일 확인
   ```
   ../examples/[브랜드명] 기획안.pdf
   ```
3. **참고 포인트**:
   - 영상 오프닝/첫 장면 연출
   - 후킹 문구 스타일
   - 썸네일 텍스트 예시
   - 궁금증 유발 방식

> ⚠️ **주의**: 기존 기획서는 **스타일 참고용**입니다. 후킹 문구나 연출을 그대로 복사하지 마세요.
